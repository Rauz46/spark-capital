## FILE SIZE INFORMATION

Most images are "scaled" versions, meaning they've been optimized for web use. Expect:
- Icons/Logos: 10-100 KB
- Illustrations: 100-500 KB
- Background Images: 500KB - 2MB
- Animated GIF: ~500KB - 1MB

Total download size: Approximately 10-15 MB for all images
